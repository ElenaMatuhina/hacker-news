## Локальная разработка

> **Заметка:**
> Команды не запустятся без установленной в системе **Nodejs**.

- Установить все зависимости можно следующими командами.

```sh
$ npm i или npm install
```

- Запустить проект можно командой.

```sh
$ npm start
```
